## About TUIkit

TUIkit is a declarative, SwiftUI-like framework for building Terminal User Interfaces in Swift. Pure Swift, no ncurses or C dependencies.

- **Website:** [tuikit.dev](https://tuikit.dev)
- **Source Code:** [github.com/phranck/TUIkit](https://github.com/phranck/TUIkit)
